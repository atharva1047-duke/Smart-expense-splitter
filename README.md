# Smart-expense-splitter
💰 Smart Expense Splitter
🚀 Overview

Smart Expense Splitter is a web application that helps users manage shared expenses within a group by calculating balances and minimizing transactions.

✨ Features
Create groups and add members
Add expenses with payer and participants
Split expenses (equal / custom)
Automatic balance calculation
Show "who owes whom"
Transaction history
Edit/delete expenses
Settle all balances
🛠️ Tech Stack
HTML5
CSS3
Bootstrap 5
JavaScript
jQuery
AJAX (JSON simulation)
localStorage
📊 Key Functionalities
Debt simplification algorithm
Real-time UI updates
Persistent data storage
Dynamic calculations
📱 UI/UX Highlights
Dashboard with balance overview
Interactive cards
Modal forms
Smooth animations
Dark mode toggle
